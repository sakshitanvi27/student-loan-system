# Student Loan Prediction System

This is a Django-based web application with an integrated machine learning model to predict student loan eligibility.

## Features
- Loan approval prediction using ML model
- Backend APIs using Django REST Framework
- Real dataset (loan_history.csv)
- End-to-end working system

## Tech Stack
- Python
- Django
- SQLite
- Machine Learning (Scikit-learn)

## Project Goal
To build a real-world AI-based financial decision support system.
